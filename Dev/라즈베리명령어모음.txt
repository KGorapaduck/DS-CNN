1. 시스템 관리 및 업데이트 (가장 중요)
라즈베리파이를 처음 설정하거나 유지보수할 때 반드시 사용하게 되는 명령어입니다.

sudo apt update: 패키지 리스트를 최신 상태로 업데이트합니다. 설치 가능한 소프트웨어의 최신 버전을 확인합니다.

sudo apt upgrade: 실제로 설치된 소프트웨어들을 최신 버전으로 업그레이드합니다.

sudo raspi-config: 라즈베리파이 전용 설정 도구입니다. Wi-Fi, 비밀번호, 인터페이스(카메라, SSH 등)를 GUI 환경처럼 쉽게 설정할 수 있습니다.

sudo reboot: 시스템을 재시작합니다.

sudo poweroff: 시스템 전원을 안전하게 끕니다. (SD 카드 손상을 방지하기 위해 권장됩니다.)

2. 파일 및 디렉토리 관리
리눅스 터미널 환경에서 파일 시스템을 다루는 기본 명령어들입니다.

## scp ds_cnn.tflite pi_virtual_mic_test.py kouy956@[IP주소]:/home/kouy956
## scp kouy956@IP주소:/home/kouy956/test.wav .
:파일 주고받기


ls: 현재 디렉토리의 파일과 폴더 목록을 보여줍니다. (ls -al은 숨김 파일까지 상세히 표시합니다.)

cd [경로]: 특정 디렉토리로 이동합니다. (예: cd /home/pi)

pwd: 현재 내가 위치한 디렉토리의 전체 경로를 출력합니다.

mkdir [이름]: 새로운 디렉토리(폴더)를 생성합니다.

rm [파일]: 파일을 삭제합니다. 폴더를 삭제할 때는 rm -rf [폴더명]을 사용합니다.

cp [원본] [대상]: 파일이나 폴더를 복사합니다.

mv [원본] [대상]: 파일이나 폴더를 이동하거나 이름을 변경할 때 사용합니다.

3. 텍스트 편집 및 파일 확인
파일의 내용을 수정하거나 확인할 때 필수적인 도구입니다.

sudo nano [파일명]: 터미널 기반의 텍스트 편집기입니다. 가장 배우기 쉽고 많이 쓰입니다.

cat [파일명]: 파일의 내용을 수정하지 않고 터미널에 바로 출력하여 보여줍니다.

grep [문자열] [파일명]: 파일 내부에서 특정 문자열을 검색합니다.

4. 네트워크 관련 명령어
라즈베리파이의 IP 주소를 확인하거나 통신 상태를 체크할 때 사용합니다.

hostname -I: 라즈베리파이의 현재 할당된 IP 주소를 바로 보여줍니다. (가장 자주 쓰임)

ifconfig: 네트워크 인터페이스(WLAN, 이더넷)의 상세 정보를 확인합니다.

ping [주소]: 특정 서버(예: https://www.google.com/search?q=google.com)와 통신이 잘 되는지 확인합니다.

ssh [사용자]@[IP주소]: 원격에서 다른 PC를 통해 라즈베리파이에 접속할 때 사용합니다.

5. 시스템 상태 모니터링 및 하드웨어
라즈베리파이 3 B+의 성능과 상태를 확인하는 명령어입니다.

htop: CPU, RAM 사용량과 실행 중인 프로세스를 시각적으로 보여줍니다. (기본 설치가 안 되어 있다면 sudo apt install htop으로 설치 가능)

df -h: SD 카드의 남은 용량을 읽기 쉬운 단위(GB, MB)로 보여줍니다.

vcgencmd measure_temp: 라즈베리파이 CPU의 현재 온도를 확인합니다. (발열 체크 시 필수)

vcgencmd get_mem arm: CPU(ARM)가 사용 중인 메모리 양을 확인합니다.

6. 패키지 설치 및 관리
새로운 소프트웨어를 설치하거나 삭제할 때 사용합니다.

sudo apt install [패키지명]: 새로운 프로그램을 설치합니다.

sudo apt remove [패키지명]: 설치된 프로그램을 삭제합니다.

sudo apt autoremove: 더 이상 필요 없는 의존성 패키지들을 정리하여 용량을 확보합니다.
